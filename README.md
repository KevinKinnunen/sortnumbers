## Instructions

- How to run:
    Run/Debug SortMain.java
 java SortMain.java
 
 The program will ask you how much numbers you want to sort, 
 example:
 3
 
 Later will you enter the numbers, 
 example:
 -30
 10
 -20

 The numbers will be sorted and printed,
 example:
 -30
 -20
 10

 - How to test:
    Run/Debug @Test in SortTest.java
